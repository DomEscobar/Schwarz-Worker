# Skill: Handoff Discipline

Use this skill when pausing/resuming work or when context pressure is high.

## Mid-plan pause protocol

1. Update STATE with milestone, phase, plan, blockers, next action.
2. Write `{phase}-CONTINUE.md` with step index and exact resume action.
3. Capture decisions with rationale, not just outcomes.
4. Capture fragile assumptions and diagnostics paths.

## Resume protocol

1. Read CONTINUE first, then STATE.
2. Execute the exact next action before re-planning.
3. Delete CONTINUE once consumed.
4. If files disagree (STATE vs plans vs summaries), stop and reconcile before coding.
