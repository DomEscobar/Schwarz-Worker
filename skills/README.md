# SchwarzWorker Skills

Optional domain skills that planner/executor/verifier can load before work.

## How to use

- Global skills can live in `skills/`.
- Project-specific skills can live in `.planning/skills/`.
- `skills.paths` in `.planning/config.json` defines lookup order.
- Keep skills narrow and practical. One skill = one repeatable domain pattern.

## Suggested naming

- `skill-<domain>.md`
- `skill-<framework>.md`
- `skill-<integration>.md`

## Included starter skills

- `skill-boundary-contracts.md`
- `skill-verification-evidence.md`
- `skill-handoff-discipline.md`
